xosField
directive in module xos.uiComponents

Description

The xos-field directive. This component decide, give a field wich kind of input it need to print.

Usage

as element:
<xos-field
       name="{string}"
       field="{object}"
       ng-model="{mixed}">
</xos-field>

Parameters

ParamTypeDetails
namestring

The field name

fieldobject

The field configuration:

{
  label: 'Label',
  type: 'number', //typeof field
  validators: {} // see xosForm for more details
}
ngModelmixed

The field value
