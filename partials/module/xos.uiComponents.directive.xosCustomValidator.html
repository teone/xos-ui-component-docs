xosCustomValidator
directive in module xos.uiComponents

Description

The xosCustomValidator directive. This component apply a custom validation function

Usage

as attribute
<ANY xos-custom-validator
     custom-validator="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
customValidatorfunction

The function that execute the validation.

You should do your validation here and return true | false, or alternatively you can return an array [errorName, true|false]
