xosAlert
directive in module xos.uiComponents

Description

The xos-alert directive

Usage

as element:
<xos-alert
       config="{Object}"
       [show="{Boolean}"]>
</xos-alert>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
configObject

The configuration object

{
  type: 'danger', //info, success, warning
  closeBtn: true, //default false
  autoHide: 3000 //delay to automatically hide the alert
}
show
(optional)
Boolean

Binding to show and hide the alert, default to true
