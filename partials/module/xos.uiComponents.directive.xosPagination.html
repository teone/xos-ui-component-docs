xosPagination
directive in module xos.uiComponents

Description

The xos-table directive

Usage

as element:
<xos-pagination
       page-size="{Number}"
       total-elements="{Number}"
       change="{Function}">
</xos-pagination>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
pageSizeNumber

Number of elements per page

totalElementsNumber

Number of total elements in the collection

changeFunction

The callback to be triggered on page change.

  • @element ANY
