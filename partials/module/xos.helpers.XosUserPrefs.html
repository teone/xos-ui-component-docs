XosUserPrefs
service in module xos.helpers

Description

This service is used to store the user preferences in cookies, so that they survive to page changes. The structure of the user preference is:

{
  synchronizers: {
    notification: {
      'volt': boolean,
      'openstack': boolean,
      ...
    }
  }
}
